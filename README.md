# Modern Alumni Admin Management System

A premium React + Tailwind CSS admin dashboard for alumni management. It includes authentication screens, protected routes, dark/light mode, reusable CRUD pages, responsive sidebar/navbar, TanStack tables, mock API, charts, modals, validation, toast notifications, skeleton states, and scalable folder architecture.

## Tech Stack

- React JS + Vite
- Tailwind CSS v4 via `@tailwindcss/vite`
- React Router DOM
- Zustand
- React Hook Form
- Framer Motion
- TanStack Table
- Recharts
- Axios
- Headless UI
- Lucide React / React Icons
- React Hot Toast

## Run

```bash
npm install
npm run dev
```

## Notes

Tailwind is configured with the official Vite plugin for modern Tailwind v4 setup. Do not use `tailwindcss` directly as a PostCSS plugin in this project.

## Main Features

- Modern responsive SaaS dashboard
- Login, register, forgot password, reset password
- Protected admin routes and simple role guard
- Sidebar with nested menus
- Dashboard analytics and charts
- Fully reusable CRUD engine
- Search, sort, pagination, CSV export, row actions, bulk delete
- Reusable modal form with validation
- Dark/light mode
- Mock API layer ready to replace with backend endpoints
- Environment variable support
